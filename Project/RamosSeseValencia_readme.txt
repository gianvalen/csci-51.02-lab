# Project: Process Scheduling created by Jade Ramos, Rafael Sese, and Gian Valencia

## Description
This program simulates various CPU scheduling algorithms using C++. It includes:
- Implementation of FCFS, SJF, SRTF, Priority Scheduling, and Round Robin
- Gantt chart generation to visualize process execution order

## How to Run
1. Compile the program using: 

g++ -o scheduler RamosSeseValencia_code.cpp

2. Run the executable:

./scheduler < test_cases.txt

3. Check the output.txt file if it has the same contents
